# NodeJS Basic setup

This is a simple basic node setup for your project and API

## Installation

```bash
On terminal run:
git clone https://github.com/chrix95/nodejs-basic-setup.git
```
```bash
On terminal run:
npm install
```

## Usage
```bash
On terminal run:
npm start
```
